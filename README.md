# 淨土宗竹山念佛會 - 法會程序表

本專案收錄「淨土宗竹山念佛會」之每週固定法會時程。本會依止 淨土宗開宗祖師 善導大師 之教譜，核心宗旨為「望佛本願，意在眾生，一向專稱，彌陀佛名」。

## 🌸 法會內容與儀軌說明

### 1. 念佛共修法會 (每週四)
本共修以「聞法」與「稱名」為核心，落實善導大師所倡之「專修」精神。
恭誦經典：誦讀《佛說阿彌陀經》，體認彌陀無條件之救度。
法義聞信：透過淨土法門講座，建立「信受彌陀救度」之正知正見，明辨善導大師「要弘二門」之別。
一向專念：安排長時段純粹念佛，實踐「一心專念阿彌陀佛名號，行住坐臥，不問時節久近，念念不捨」之正定業。

### 2. 念佛超薦法會 (每週日)
本法會以念佛功德，祈願彌陀本願加持，令亡者往生、生者安心。
本願稱名：不論善惡罪福，唯藉「彌陀名號」之不可思議功德，作為超薦之唯一依靠。
法語開示：引導大眾體悟「自身現是罪惡生死凡夫」，進而「一心歸命」彌陀救度。
慈悲迴向：將念佛功德平等施予眾生，祈願同生安樂國。

## 📍 關於淨土宗竹山念佛會
指導思想：善導大師之淨土思想。
特色：不雜餘修，專門、純粹、易行。
法門宗旨：信受彌陀救度，專稱彌陀佛名，願生彌陀淨土，廣度眾生。
